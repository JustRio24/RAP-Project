## 📖 Overview

**KONSTRA** is a progressive web application (PWA) designed to streamline contractor management, site operations, and project monitoring. Built with a modern tech stack, it provides an offline-first experience suitable for remote site usage, ensuring productivity even in low-connectivity areas.

## ✨ Key Features

- 📱 **Offline-First PWA** - Installable on desktop and mobile devices. Fully functional offline with robust service worker caching.
- 📊 **Dashboard & Analytics** - Real-time insights into project progress, resource allocation, and budget tracking.
- 👥 **Team & Payroll Management** - Comprehensive worker registry, attendance tracking, and payroll generation.
- 📄 **Advanced PDF Reports** - Generate polished project reports and invoices on the fly using `jspdf` and `@react-pdf/renderer`.
- 🎨 **Premium UI/UX** - Built with Tailwind CSS v4 and Radix UI primitives for a sleek, corporate-grade aesthetic.

## 🛠️ Technology Stack

- **Framework:** React 19 + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4 + Lucide Icons
- **State Management:** Zustand
- **Routing:** React Router v7
- **Forms & Validation:** React Hook Form + Zod
- **Components:** Radix UI

## 🚀 Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed (v20+ recommended).

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/JustRio24/KONSTRA.git
   cd KONSTRA
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## 📱 PWA Features

This application utilizes `vite-plugin-pwa` for progressive web app features:
- Prompt for update when new content is available
- "Add to Home Screen" support on iOS and Android
- Offline availability of core assets
